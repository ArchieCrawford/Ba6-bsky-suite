# Scheduling

## Draft to scheduled post

1. Create or select a draft.
2. Choose a scheduled time.
3. Optionally pick a connected Bluesky account.

## Queue statuses

- **Queued**: waiting to run
- **Posting**: currently being sent
- **Posted**: successfully published
- **Failed**: error occurred
- **Canceled**: removed from the queue

## Tips

- Keep an eye on failed jobs in **Events**.
- You can re-run or cancel a scheduled item from **Scheduled**.
